# Node Weather CLI App

### About

Weather forecast in the console for the specified address. Project for Udemy NodeJS course.

### Requirements

[Node](https://nodejs.org/)

### Install

```
git clone git@github.com:ermondel/node-weather-app.git
```

```
npm install
```

### Usage

```
npm run app Kiev
```

### Technologies

- Node
- request
- darksky.net api
- mapbox.com api
